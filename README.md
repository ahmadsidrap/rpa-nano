## Installation

### Step 1 - DB migration

```bash
python manage.py migrate
```

### Step 2 - Create superuser

```bash
python manage.py createsuperuser
```

### Step 3 - Run the server

```bash
cd rpaNanoRouter
python manage.py runserver
```
