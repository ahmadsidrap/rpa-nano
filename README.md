# rpa-nano
RPA for nano services

### Official homepage

[https://introvesia.com](https://introvesia.com)

## RPA Router

RPA Router is an API to handle request.

### Run the server

```bash
cd rpaNanoRouter
python manage.py runserver
```

### DB migration

```bash
python manage.py migrate
```

### Create superuser

```bash
python manage.py createsuperuser
```
